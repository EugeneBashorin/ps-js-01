&& - Логическое «И»

Оператор && приводит все операнды к булю и возвращает значение одного из них.
Левый операнд если его можно привести к false, и правый в остальных случаях.
выражение && выражение
Если хотябы один из операндов будет приведён к false, результатом выражения будет его значение.
То есть логическое «И» запинается на лжи и возвращает то на чём запнулось или последний операнд.
При выполнении логического «И», правый операнд может не вычисляться, в случае если левый был приведен к false.

console.log(5 && 6 && 7 && 'hello') => 'hello'
console.log(false && 'hello') => false
console.log(0 && 'hello') => 0
console.log(5 && 7 && 0 && 'hello') => 0

Логическое "И" - "&&"
- Запинается на лжи
- Возвращает то, на чем запнулось или последий операнд.

|| - Логическое «ИЛИ»

Оператор || приводит все операнды к булю и возвращает значение одного из них.
Левый операнд если его можно привести к true, и правый в остальных случаях.
выражение || выражение
Если хотябы один из операндов будет приведён к true, результатом выражения будет значение true.
То есть логическое «ИЛИ» запинается на правде и возвращает то на чём запнулось или последний операнд.
При выполнении логического «ИЛИ», правый операнд может не вычисляться, в случае если левый был приведен к true.

console.log(5 || 6 || 7 || 'hello') => 'hello'
console.log(false || 0 || null || 9 || 'hello') => 9
console.log(false || 0 || null) => null

Логическое "ИЛИ" - "||"
- Запинается на правде
- Возвращает то, на чем запнулось или последий операнд.

! - Логическое «НЕ»

Логическое «НЕ» это унарный оператор - выполняющий операцию над одним операндом справа.
Оператор ! приводит операнд к булю, если необходимо, а затем делает инверсию - заменяет его на противоположный true -> false или false -> true.

Ложные преобразования

Запомните 6 ложных (falsy) значений, приводящихся к false в логическом преобразовании: 0, NaN, null, undefined, пустая строка и false. Абсолютно все остальное приводится к true.

console.log(!0) => true
console.log(!1) => false
console.log(!"qwe") => false
console.log(!false) => true
console.log(!true) => false

Логическое "НЕ" - "!"
- Не оператор
- Делает инверсию правда > ложь и ложь > правда.
